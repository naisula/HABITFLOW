# HabitFlow

HabitFlow is a simple habit tracking program developed by python
HabitFlow helps users to:
- create habits 
- track daily tasks
- store progress in database
- manage productivity routines

## features

add new habits
view habits
add new tasks 
view tasks
store progress in database

## technologies used
- python
- sqlite3
- streamlit
- git 

## project structure

main.py         # starts the application / terminal version of application
database.py     # connects to sqlite database
habits.py       # handles habit related logic
tasks.py        # handles tasks
goals.py        # handles habit goals
app.py          # handles the connection to streamlit
